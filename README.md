# ClassSitDown

班级 MP 发放助手。

用于老师登录教务系统、选择班级、导入数据文件、核对发放明细，并导出结果。

## 目录

- `frontend/`: React + Vite 前端
- `backend/`: Express + Puppeteer 后端
- `使用说明.html`: 操作说明页

## 本地启动

### 1. 启动后端

```bash
cd backend
npm install
npm run dev
```

### 2. 启动前端

```bash
cd frontend
npm install
npm run dev
```

## 构建前端

```bash
cd frontend
npm install
npm run build
```
